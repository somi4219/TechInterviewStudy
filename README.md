## ✅ 진행방식

1. 마지막 주 ~요일 : 다음 주제 키워드 선정 및 키워드에 맞춘 일정 설정
2. 매주 ~요일까지 PR 올리기
    
    ex ) `Network/Somi/ApplicationLayer/README.md`
    
    `Network/ChaeHyun/ApplicationLayer/README.md`
    
    PR 명 :     [ ADD ] 주제 키워드명 - 이름
    
3. 매주 ~요일까지 PR 서로 피드백 , 확인 후 merge
4. ~주에 1번 인당 10분씩 모의면접
    - 키워드에 대한 질문 미리 준비
    
    면접관 → 면접자 : 서기
    
    서기가 면접 내용 + 서로 해준 피드백 정리해서 면접자 폴더에 PR 올리기
    
    ex ) `Network/Somi/INTERVIEW.md`
